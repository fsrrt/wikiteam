The log with all the news related to WikiTeam.

## 2012 ##
  * 2012-11-09: Specifications are written for a [rewrite of dumpgenerator.py](https://meta.wikimedia.org/wiki/WikiTeam/Dumpgenerator_rewrite) to fully benefit of MediaWiki WebAPI (probably via pywikipediabot framework).
  * 2012-08-06: [Uploader](http://code.google.com/p/wikiteam/source/browse/trunk/uploader.py) script is finally coded and we start to make some test uploads before people launch it to upload the thousands TaskForce downloaded dumps.
  * 2012-06-07: The SpeedyDeletion tool has been running for some weeks collecting English Wikipedia articles to be deleted every 30 minutes. The articles are being saved at [Internet Archive](http://archive.org/details/wikipedia-delete-2012-06) and [speedydeletion.wikia.com](http://speedydeletion.wikia.com).
  * 2012-05-17: Wikimedia projects dumps are now mirrored in [three different locations](http://dumps.wikimedia.org/mirrors.html) around the globe. Very good news.
  * 2012-04-25: Steven Leeman archives [Fon wiki](http://wiki.fon.com) ([WebCite](http://www.webcitation.org/67BNif6yE)) and uploads it to [Internet Archive](http://archive.org/details/wiki-wikifoncom_563). Thanks!
  * 2012-04-07: We start a TaskForce to archive 7,000+ wikis from a list by Andrew Pavlo (2009).
  * 2012-04-05: WikiTeam 1st Birthday! \o/
  * 2012-03-03: WikiTeam tools need a Graphical User Interface ([GUI](GUI.md)) to reach a greater community. We start to work on this.
  * 2012-02-27: We start to [archive Wikimedia Commons](http://www.archiveteam.org/index.php?title=Wikimedia_Commons). [Source code](http://code.google.com/p/wikiteam/source/browse/trunk/commonsdownloader.py) and [file lists](http://toolserver.org/~emijrp/commonsarchive/) available. We are doing some tests.
